## Requirements Overview

### Scope

- **Egypt-only tours** — packages, day tours, Nile cruises
- **Multilingual** — English, Spanish, Arabic (RTL)
- **Dual booking modes** — Instant booking + Request-to-book
- **Content platform** — Blogs, vlogs, media galleries

### Methodology

Requirements follow the MoSCoW prioritization:

| Priority | Meaning |
|----------|---------|
| **Must** | Core functionality, launch blocker |
| **Should** | Important but not blocking |
| **Could** | Nice to have, Phase 2+ |
| **Won't** | Explicitly out of scope |
