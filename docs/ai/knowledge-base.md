# Knowledge Base

## Data Sources

| Source | Content |
|--------|---------|
| Trip Database | All published trips |

| Destination Info | Egypt travel info |
| Policies | Booking, cancellation |

## RAG Pipeline

1. User query received
2. Query embedded
3. Similar documents retrieved
4. Context assembled
5. LLM generates response
6. Response validated and returned
