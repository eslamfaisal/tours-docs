# Analytics

## Available Reports

| Report | Description |
|--------|-------------|
| Booking Report | Bookings by date, status, destination |
| Revenue Report | Revenue by period, trip, channel |
| Customer Report | New vs returning, demographics |
| Trip Performance | Bookings per trip, conversion |
| Traffic Report | Sources, referrals, pages |
