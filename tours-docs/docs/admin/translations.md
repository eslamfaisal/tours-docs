# Translation Management

## Workflow

1. Content created in English (default)
2. Translation tasks created for ES/AR
3. Editor translates content
4. Review and approve
5. Publish synchronized

## Translation Status

| Status | Description |
|--------|-------------|
| Pending | Not started |
| In Progress | Being translated |
| Review | Awaiting approval |
| Approved | Ready for publish |

## Translatable Content

- Trips (title, description, itinerary)
- Destinations
- Blog posts
- Static pages
- UI strings
