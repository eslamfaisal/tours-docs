# Facebook Migration

## Content to Migrate

| Content Type | Source | Target |
|--------------|--------|--------|
| Posts | Facebook Page | Blog |
| Videos | Facebook Videos | Vlogs |
| Photos | Photo Albums | Media Library |
| Reviews | Page Reviews | Reviews (imported) |

---

## Migration Steps

1. Export Facebook data via Page Insights or manual
2. Download media assets (photos, videos)
3. Transform content to blog/vlog format
4. Upload media to CDN
5. Create content entries in CMS
6. Review and publish
